作品集照片放这里（作品页 work.html 用）
=======================================

按招牌类型命名，丢进这个 images/work 文件夹。每类先放了 6 个位置，
想加更多张，照着往下编号（led-7.jpg、led-8.jpg…）再告诉我，我帮你在页面加格子。

3D LED 发光字招牌：     led-1.jpg     led-2.jpg … led-6.jpg
3D 立体招牌：           3d-1.jpg      3d-2.jpg  … 3d-6.jpg
3D 不锈钢招牌：         steel-1.jpg   steel-2.jpg … steel-6.jpg
普通招牌：              normal-1.jpg  normal-2.jpg … normal-6.jpg
霓虹灯招牌：            neon-1.jpg    neon-2.jpg … neon-6.jpg
室内招牌：              indoor-1.jpg  indoor-2.jpg … indoor-6.jpg

- 横图（4:3）最好看。没放图的格子会显示文件名占位，放上就自动变照片。
- 「查看更多作品」按钮（首页）会跳到 work.html 这个页面。
